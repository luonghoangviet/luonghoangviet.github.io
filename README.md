# luonghoangviet.github.io

Static GitHub Pages layout with three khu vực:

- Portfolio / CV: tóm tắt và link tải CV.
- Bài viết kỹ thuật: danh sách bài tech.
- Linh tinh: ghi chép không chuyên.

## Cách dùng nhanh

1) Đặt file CV tại `assets/Viet-Hoang-Viet-CV.pdf` (hoặc đổi đường dẫn trong `index.html`).
2) Sửa nội dung thẻ trong `index.html` cho đúng thông tin (email, mô tả dự án, liên kết).
3) Thay dữ liệu mẫu trong `posts/index.html` bằng link thật tới từng bài.
4) Push lên branch `main` để GitHub Pages build (Settings → Pages → Deploy from branch → `main` / root).

Màu sắc, font và bố cục được định nghĩa trong `assets/css/style.css`.
